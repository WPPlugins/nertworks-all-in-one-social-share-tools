== Description ==

NertWorks Social Share Tools make it easy for you to place social share and like buttons through out your website.  Currently supports Addthis, Addtoany, as well as Custom Social bars and buttons.  


== Installation ==
1. Upload `nertworks-social-share-tools` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Customize your buttons the way you want.  
4. Share your site til the cows come home.
